2. 对首页做更明确的作者品牌化升级

## 3. 首页升级设计

### 3.1 方向

采用“作者品牌型，但别太花”的路线。

### 3.2 强化目标

首页需要更明确表达：

- 作者是谁
- 当前处于什么技术阶段
- 博客主要关注什么主题

### 3.3 升级点

1. Hero 区
   - 更明确表达作者身份与转型方向
   - 增强层次感，但不做夸张视觉

2. 作者卡片
   - 更像个人名片
   - 仍保持辅助位置，不压过文章区

3. 文章区
   - 卡片更稳、更精致
   - 入口更明确

4. 主题区
   - 更像“长期写作方向入口”
   - 不只是简单标签陈列

### 3.4 不做项

- 不加重动画
- 不加过强渐变
- 不改成作品集 landing page
- 不牺牲技术博客的阅读重心

## 4. 验收标准

完成后应满足：

1. About 页第一屏可以快速识别作者身份与联系方式
2. 首页第一屏可以清楚传达作者身份与博客定位
3. 首页比现状更有辨识度，但整体仍然克制
4. Hugo 构建通过

## 5. 结论

本轮不是重做结构，而是在现有成熟骨架上做一轮更明确的“作者表达强化”。
